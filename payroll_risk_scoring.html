from sklearn.metrics import accuracy_score, precision_score, recall_score

models = {
    "Logistic Regression": lr,
    "Random Forest": rf,
    "Gradient Boosting": gb
}

results = []

for name, model in models.items():
    y_pred = model.predict(X_test)
    results.append([
        name,
        accuracy_score(y_test, y_pred),
        precision_score(y_test, y_pred),
        recall_score(y_test, y_pred)
    ])

results_df = pd.DataFrame(
    results,
    columns=["Model", "Accuracy", "Precision", "Recall"]
)

Feature Importance Analysis (Markdown)

The feature importance analysis highlights the key factors contributing to payroll risk. Behavioral indicators such as employee performance rating, engagement score, satisfaction score, and work-life balance score play a significant role in identifying high-risk employees. This insight helps HR teams understand which factors require attention to reduce payroll and workforce instability.

Model Selection and Justification

Three machine learning models were trained: Logistic Regression, Random Forest, and Gradient Boosting. Among these, the Random Forest model performed best due to its ability to handle non-linear relationships and interactions between employee behavior indicators. It provides better generalization and interpretability through feature importance analysis, making it suitable for payroll risk scoring.

Conclusion

This project successfully builds a Payroll Risk Scoring System using employee performance and engagement data. Feature engineering was applied to derive payroll risk indicators, and machine learning models were used to predict high-risk employees. The system can assist HR and payroll teams in proactively identifying employees requiring attention, thereby improving workforce stability and payroll planning.
