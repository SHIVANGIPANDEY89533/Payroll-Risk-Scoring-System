# Payroll Risk Scoring System

## Project Overview
This project analyzes employee performance and engagement data to identify payroll risk factors using machine learning techniques.

## Tools & Technologies
- Python (Pandas, Scikit-learn, Matplotlib)
- Jupyter Notebook
- CSV Dataset

## Key Features
- Feature Engineering
- Payroll Risk Classification
- Model Comparison (Logistic Regression, Random Forest, Gradient Boosting)
- Feature Importance Analysis

## Files Included
- dataset.csv – Employee dataset
- Payroll_Risk_Scoring_System.ipynb – Model development
- payroll_risk_scoring.html – Output visualization
- Documentation.docx – Project documentation
